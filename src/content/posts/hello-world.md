## 为什么开始写博客？

一直想拥有一个属于自己的独立博客空间，在这里可以自由地表达想法，记录技术探索的过程，分享学到的知识。终于在 2026 年的夏天，这个博客上线了 🎉

写博客有几个好处：

1. **整理思路** — 把零散的想法写成文章，能帮你更好地梳理知识体系
2. **分享价值** — 你踩过的坑、解决的问题，可能正好帮到另一个人
3. **积累沉淀** — 时间久了回头看看，能看到自己的成长轨迹

## 技术选型

这个博客使用了以下技术栈：

- **Astro** — 极快的静态网站框架，支持 SSR
- **TinaCMS** — 在线可视化编辑，写完就能发布
- **Giscus** — 基于 GitHub Discussions 的评论系统，登录即关联 GitHub 账号
- **Upstash Redis + Vercel Serverless** — 文章点赞功能，全球低延迟
- **Vercel Blob** — 文件上传与存储
